# ScreenRatio

Outil en ligne permettant de calculer le ratio 16:9, 21:9 ou 4:3 à partir de la largeur connue.

## Demo

[https://cedricbeau.github.io/ScreenRatio/](https://cedricbeau.github.io/ScreenRatio/)
